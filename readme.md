Rozważana jest możliwość dostaw węgla kamiennego z trzech kopalń A, B, C do trzech elektrowni
F, G, H za pomocą sieci kolejowej z dwoma stacjami pośrednimi D i E.

(koszt, przepustowość):

A       3,8         6,10    -       -       -
B       6,10        3,13    -       -       -
C       4, 10       5,8     -       -       -
D       -           2,20    5,16    7,6     3,10
E       -           -       5,7     4,4     2, 2
